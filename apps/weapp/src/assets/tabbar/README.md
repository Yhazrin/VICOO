# WeChat Tab Bar Icons

当前为占位图标（1x1 透明 PNG）。

## 替换为真实图标

请使用设计工具导出以下图标，规格：

- 尺寸：81x81 px（建议）
- 格式：PNG，带透明背景
- 普通态颜色：#9CA3AF（灰色）
- 选中态颜色：#1A1A1A（黑色）

| 文件名 | 对应 Web 图标 | 含义 |
|--------|-------------|------|
| home.png / home_active.png | Material Icons `home` | 首页 |
| inbox.png / inbox_active.png | Material Icons `inbox` | 收件箱 |
| ai.png / ai_active.png | Material Icons `smart_toy` | AI 助手 |
| library.png / library_active.png | Material Icons `library_books` | 知识库 |
| profile.png / profile_active.png | Material Icons `account_circle` | 我的 |

可从 https://fonts.google.com/icons 下载 Material Icons 的 PNG 版本。
