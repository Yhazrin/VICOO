## ‚ú® Philosophy: What is "Vibe Coding"?

Traditional note-taking apps are sterile archives. **vicoo** is designed to be:

1.  **Alive**: Animated mascots react to your work state. It feels like a collaboration, not data entry.
2.  **Spatial**: Knowledge is a galaxy, not just a list of files.
3.  **Proactive**: The "Neural Gardener" suggests connections before you realize them.
4.  **Tactile**: Bold borders, pop colors, and satisfying interactions make knowledge work feel like a game.

## üé® Design System: "Neo-Lite"

vicoo uses a custom design language built on **Tailwind CSS**:

| Palette | Hex | Usage |
| :--- | :--- | :--- |
| **Primary** | `#FFD166` | Highlights, Success, Energy |
| **Secondary** | `#0df259` | "Vibe" Green, Confirmations |
| **Accent** | `#EF476F` | Pop Pink, Creative actions |
| **Info** | `#118AB2` | Deep Blue, Technical data |
| **Ink** | `#1a1a1a` | High Contrast Text & Borders |

*   **Shadows**: Hard-edge CSS shadows (`box-shadow: 4px 4px 0px #1a1a1a`).
*   **Borders**: Thick 3px borders for a tactile feel.
*   **Typography**: *Plus Jakarta Sans* (UI) & *Space Grotesk* (Display).
